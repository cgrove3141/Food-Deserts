API Documentation

All API routes return the following information by census tract:

  • Cenusus Tract
  • Zip Code
  • Food Desert Status (defined as low income and low access tract measured at 1 mile for urban areas and 20 miles for rural areas)
  • Congressional District
  • Contact Info:
    • Representative
    • Phone #
  • Population
  • Link to County Map by Census Tract

The following routes can be used:

  • /api/v1.0/zipapi/[zip_code] to return all tracts in a given zip code


  • /api/v1.0/stateapi/[state] to return all tracts in a given state


  • /api/v1.0/tractapi/[census_tract] to return all tracts by tract number (there may be multiple values returned, as there can be several zip codes/congressional districts per tract.

Additionally, by removing "api" from route, a user can access JSON style data as non-callable text within the confines of the webpage (e.g. /api/v1.0/zip/[zip_code])
